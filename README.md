# My Cypress automation

this project utilizes cypress to automate the testing of a login feature and a user update feature.
Cypress is a javaScript-basead end-to-end testing framework that allows developers to write tests and
execute them in a real browser.

## Installation
```bach 
npm install 
```
>**NODE:**
>
> it is necessary to have **node** installed

## Running
```bash
# You open the cypress UI
npx cypress open

# Run the automation by command line
npx cypress run
```
